# Blocksack Launcher
A lightweight, open-source non-premium Minecraft launcher written entirely in Python by Fitzypopper. 

## Features
* Bypasses heavy bloatware - pure Python execution.
* Full support for different isolated instances (folders).
* Multi-threaded asset downloading.
* Simple Tkinter GUI.

## How to run
1. Ensure Python 3 and Tkinter are installed (`sudo apt install python3-tk`).
2. Run `python3 menu.py`.
